一.基础

1.头文件导入

   导入方法 #include <标准库名>  或 #include "自定义库名"
   math.h	stdio.h	stdlib.h	string.h
注意:
      1.每一个.c文件的开头一定要有头文件<stdio.h>
      2.一个项目只能有一个main()方法

2.数据类型:

 1.整形: int,short,long,unsigned short,unsigned  int,unsigned long
 2.浮点型: float,double
 3.字符型: char,unsigned char
 4.无值型: void
 5.指针类型:
 6.构造类型: 数组 结构体 联合体
//字节数(64位操作系统):char 1;  short 2;  int 4;  long 8;
					 float 4;  double 8;  指针 8;
//别名创建格式: typedef 类型 别名;   (允许嵌套,且一个类型可以有多个别名)

3.常量

定义以后就无法改变的量
  *定义符号常量: #define 常量名 常量值
  *字符型常量: 用单引号 '5'储存的是其对应的ASCII码 53(D)           小写字母的ASCII码比大写字母的ASCII码大32
  *字符串常量: 用双引号
  十进制整型常量无开头
  八进制整型常量以0开头
  十六进制整型常量以0x开头
  长整型常量以L做后缀
  无符号常量以U做后缀

4.转义字符

\n是换行   \t是制表   \r是回车    \b是退格   \\是反斜线
\'是单引号   \“是双引号
转义号\+字符对应ASCII码可以表示对应字符常量

5.运算符

优先级: 单目>双目
  *自增自减 y=x++ 为先赋值后x+1    y=--x 为先x-1后赋值

  *逗号运算符: 用来连接多个表达式
             优先级最低,结合性 从左到右,且以最后一个表达式的值为结果
  *赋值复合运算符:
          优先级仅仅大于逗号运算符
          结合方向位 从右到左
          格式: 变量名 算术或位运算符= 表达式(表达式是一个整体的值)  变量名 = 变量名 算数或位运算符 值;        
  *赋值语句
       变量名=表达式
       变量名1=变量名2=....=表达式
       (变量名1,变量名2,....)=(变量名1,变量名2)
       结构名2=结构名2
       结构名={值1,值2,.......}
       变量名=条件表达式?表达式T:表达式F

1.类型转换: 低转高自动转 高转低要强转  格式为: (类型符)表达式;

四.函数的定义和调用

1.定义

分类: 库函数和自定义函数
  声明: 类型符 函数名(参数类型1,参数类型2,.......);           声明可以和定义变量放一起
  定义: 类型符 函数名(定义形参1,定义形参2,.......){};         ******函数调用前一定要声明

2.调用: 函数名(实参);

函数永远时单项值传递,且传入的实参中有表达式时,先从右向左算,再从左向右传入
  //函数多返回值可以用指针和全局变量进行操作
  函数递归的要求: 终止条件 递归规则

五.变量的作用域和存储类别

1.作用域为变量起作用的范围

分类:  局部变量->函数体或符合语句(语句块)内有效
      全局变量->定义在函数之外,在定义后的函数中可以调用,但在局部中不可用

2.存储类别

分类: 静态-->定义之后就一直存在,直到程序结束  用static修饰
     动态-->用完就消失
关键字: auto 自动存储
       register 寄存器
       static 静态 (对于为赋于初值的变量,默认赋值为0)  不能跨文件调用
       extern 外部   可以使在主函数外定义的变量在主函数中可以调用
注意:   变量储存类型
       局部变量定义后未初始化时输出为随机值
       初始化定义变量时默认为auto(动态储存类型)
       register(寄存器)型变量运算速度比auto快
       全局变量为定义在所有函数之外的变量

六.预处理命令

1.宏定义

关键字: define
  无参定义句法: #define 标识符 字符序列
  有参定义句法: #define 标识符(参数列表) 字符序列
  注意: 宏名不可以作为赋值对象  宏定义是将程序中宏名替换成对应的字  符序列  计算式尤其注意括号  标识符一般大写  宏定义没有赋值号和结尾号

2.文件包含

关键字: include
包含句法: #include <文件名>    只在系统规定的路径中寻找
         #include "文件名"    在当前文件中寻找

3.条件编译

1.#ifdef 标识符1
	程序段1
  [#else
	程序段2]
  #endif
  作用:当标识符1被宏定义后,编译程序段1,否则编译程序段2
2.#ifndef 标识符2
  	程序段1
  [#else
	程序段2]
  #endif
  作用:当标识符2没有被定义时,编译程序段1,否则编译程序段2
3.#if 逻辑量
	程序段
  [#else
  	程序段]
  #endif
  作用:当逻辑量为真时,编译程序段1,否则编译程序段2

七.数组

1.一维数组

定义语句: 类型符 数组名[数组长度];数组第一个数据的下标为0
初始化语句: 类型符 数组名={}
引用语句: 数组名[下标]

2.二维数组

定义语句: 类型符 数组名[行数][列数]   
		存储顺序为一行一行的  相当于每一个一维数组的数据存放了另一个一维数组
初始化语句: 类型符 数组名={{},{},{},{}}        存储一一对应顺序时可以省略一维数组的花括号
          部分特定设初值时 其余都默认赋值为0
          全部设初值时常量1可以省略但常量2不可以
引用语句: 数组名[下标1][下标2]
注意: 在输出数组时不能添加换行符

3.特点

数组可以作为函数形参，此时数组相当与指针
数组数据传入形参时是地址传递  形参的变化为实参有同样影响
数组变量本身就是一个指针 表达地址

八:指针

1.指针变量:用来存放目标变量的地址

定义: 目标变量类型符 *指针变量名 = 地址值(&+目标变量名)    若地址值为0或NULL,则为空指针
  注意: 赋值时指针变量名 不带*号  且*号只对最近的变量有效
       指针不能赋值常量 指针本身的值时地址值
const在星号左边，如const char *p，则const修饰类型符,指针指向的内容不能改变
const在星号右边，如char *const p，则const修饰指针,指针指向不可改变
const在星号左边和右边，如const char const *p，表示p的指向不能改变，指向的变量内容也不能改变。

2.运算符

取地址运算符:    &用来获取目标变量的地址
直接访问运算符:  *获取目标变量的内容
运算符的关系例如下:
int i,*p=&i;          p &i &(*p)等价            i *p *(&i)等价
//对指针进行+(-)运算时 指针向后(前)移动n个对应类型的字节数
//指针之间可以做减法 p1-p2的值为两地址之间数据的数量

3.指针一维数组

定义: 目标类型符 (*指针名)[数组长度] = &数组名
指针地址对应关系:  例如int (*p)[5]=&a,a[5];
                a=p=&a[0]    &a[i]=p+i=a+i
对于数组值的引用:
                a[i]=*(p[i])=*(a+i)

//p指向数组a[n]时 p[m]=a[n+m] m<0时且下标未越界仍有效

4.指针二维数组

// 地址对应关系:  例如int a[i] [j];
行指针定义: 一维数组指针可以指向二维数组的某一行
          等价地址: a+i=&a[i]
列指针定义: 等价地址: a[i]+j=&a[i][j]=*(a+i)+j
          等价内容: *(a[i]+j)=a[i][j]=*(*(a+i)+j)

5.指针数组

定义:   类型 *指针数组名[长度];
//数组中每个单元都是用来存放地址的

6.指针函数

1.指针做形参
  类型符 函数名(*指针名, .....){}
  此时传入函数形参的值一定要是地址值,在该函数内可以简洁的访问传入的全局变量
  数组就是一个特殊的指针
  ### 指针传入函数时,形参和实参指向同一个目标,对实参和形参的影响都对目标起作用
2.指向函数指针
  定义: 返回值类型 (*指针名)(形参类型表列)=函数名;
  //函数入口地址是函数体第一条指令的地址,可以用函数名引用
  引用:函数指针()=(*函数指针)()=函数名()=(*函数名)()=(&函数名)()
3.返回指针值的函数
  定义: 类型符 *函数名(形参){代码块}

7.指针与动态内存管理

//调用相应函数需要预处理头文件<stdlib.h>

九.处理字符串

字符串被存放时会默认在其最后添加结束标志符\0 (非有效字符，但是该标识符占一个字节存储单元)

1.用字符数组表示字符串

一维字符数组初始化: 1.char 数组名[长度]={'','',''.........,}            
                 2.char 数组名[长度]={"字符串"}     {}可以省略
字符串变量初始化:  char 字符串名[最大长度]=".....\0";  
//当最大长度大于实际长度时\0可以省略    
***初始化后 字符数组只能按元素赋值
		   字符串不能赋值,只能通过字符串处理函数进行操作
***若以指针来存放字符串,则可以在初始化后进行赋值,保存字符串的首地址

二维字符数组初始化: 1.char 数组名[行数][列数]={{'',..'\0'},{'',..'\0'}...}
                 2.char 数组名[行数][列数]={"字符串",字符串"...}{}可以省略
打印方式: printf(字符串数组名);printf("%s",字符串变量);
//若多个字符串之间没有任何符号,将会被自动连接
//字符串本身是常量指针,也是一个字符数组
//字符数组不等价于字符串
  1.字符串有标记\0
  2.字符串不能进行赋值,只能通过string.h库函数来处理修改字符串
  3.字符串可以输入,相当于一个指针,输入时为替换目标位置之后的字符

2.字符型指针

定义: char *指针名;
初始化: char *指针名=&字符变量  或 char *指针名="字符串"
    ***注意字符型指针不能指向字符常量,但是可以指向字符串常量,其代表的时字符串的首地址
    ***用指针直接指向字符串时不能改变字符串本身的内容
    ***利用指针输出字符串是输出指针指向的字符后的所有字符

3.字符串的输入和输出

1.scanf("%s",字符串数组名)    
  //输入的字符串中的空格 制表符 回车都将被看作字符串的结束符
  //% s之间可以加整数n来限制读入的字符长度
  printf("%s",字符串数组名)
  对于scanf()函数可以用此格式来自定义结束符  scanf("%[^  ]",str1)
2.字符串专用的输入输出函数
  输入: gets(字符串数组名)   //用回车来结束输入
  输出: puts(字符串数组名 数组的指针名 字符串常量)   //自动换行

4.常用的字符串处理函数库<string.h>

常用的字符串处理函数库<string.h>
     1.求字符串长度函数 strlen(str)     //字符串所占字节数=字符串长度+1
     2.复制字符串函数 strcpy(str1,str2或指针)  ---->将str2的字符串复制给str1
                                            ---->当str2为指针时可以移动指针以达到复制指针位置以后的字符
                    strncpy(str1,str2,n)    ---->可以将str2的前n个字符复制给str1 但是此复制不自动包括结束符\0
                    当n>str1的长度时会自动补充结束符
     3.连接字符串函数 strcat(str1,str2)   ---->将两个字符串连接      
     4.字符串比较函数 strcmp(str1,str2)   //返回值为逻辑值1或0
       对两字符串从左到右按对应字符的ASCII码值的大小进行比较,直到出现不同或\0
     5.字母大小写转换符号 strlwr(str)  大变小
                       strupr(str)  小变大
     6.字符串查找函数  strstr(x,y)   在字符转x里查找字符串y的位置

十.构造数据类型（相当于java的类）

1.定义结构体

关键字struct
struct 结构体类型名
{
 类型标识符 成员名;
 类型标识符 成员名;    //即属性
       .
       .
       . 	
 或者 struct 另一结构体名 成员名
};

2.定义结构体变量

1. struct 已经定义的结构体名 变量名1,变量名2 .....; 
   //如此结构体被定义的属性 都统一赋值给结构体变量1和2了
   struct 结构体类型名{成员变量}结构体变量名1,2......;
   //定义结构体类型时不能初始化
2. 定义别名
   1.typedef struct 已经定义的结构体名 别名;
   2.typedef struct{成员变量...........}别名;                  //别名一般首字母大写
   ****由此,再次定义结构体变量时可以简化语句                      别名 变量名1,变量名2......
   但是定义别名不是创建新的结构体
3.结构体变量初始化
   别名 变量名 = {值1,值2,...}    
   //这里的值要与成员名一一对应
   //初始化时各个成员按顺序初始化,各个数据项之间用逗号隔开

3.结构体变量的引用

1.结构体变量名.成员名       //除开初始化,对属性赋值只能一一赋值
2.同类型结构体变量之间可以整体赋值   结构体变量1=结构体变量2
3.结构体变量可以作为函数参数  //传入时为一个整体

4.结构体数组

struct 结构体类型名 数组名[常量];
初始化:*相当与对二维数组的初始化*
      struct 结构体类型名 数组名[常量]={{属性1,属性2.....},{},{},{}............}
引用成员: 结构体数组名[下标].成员名;
当结构体数组作为函数形参时,函数体内对形参的操作会影响外面传入的实参

5.结构体指针变量

定义: struct 结构体名 *结构体指针变量名;
初始化: struct 结构体名 *结构体指针变量名=&结构体变量          （用指针引用的是结构体变量中第一个成员）
引用成员: 1.(*指针名).成员名
         2.指针名->成员名   //指向运算符->
当结构体指针变量作为函数形参时,函数体内对形参的操作会影响实参
将数组赋值给指针:   结构体指针=结构体数组名; //赋值结果为首地址
***且指针移动是对数组下标的移动

十一.共用体类型

1.定义: union 共用体类型名{类型标识符 成员名1,.....};
2.共用体变量定义: union 已定义的共用体 变量名1,2,.....
  union 共用体类型名{类型标识符 成员名1,.....}变量名1,2.....;
3.共用体变量的初始化: 
  union 已定义的共用体 变量名={第一个成员的值}           
  //只能定义第一个成员
4.共用体与结构体的区别
  共用体中的所有成员共用一个存储单元,该存储单元是最大的成员所属的单元,每次给共用体中的成员赋值

十二.枚举类型

定义: enum 枚举类型名{枚举元素名1=初值,枚举元素名2,........}
//枚举元素是常量,不能更改它们的值,系统默认赋值依次为0 1 2 .....
枚举类型变量定义: enum 已定义的枚举类型 变量名
//枚举类型变量常用于循环变量的条件,枚举类型常量常用于多分枝选择结构swith

十三.链表

1.链表的结构

链表是由一系列节点组成,一个节点由指针域和数据域组成
头节点head 和 尾节点last 的指针域都为空

2.链表节点的类型定义

节点本质是结构体类型,包含若干成员
格式: struct 节点类型名{数据成员定义;struct 节点结构体类型名 *指针变量名;}

3.创建链表

插表头算法:
	1.给头指针head赋值NULL
	2.建立新节点,并把该节点的地址赋给指针变量p
	3.如果head为NULL,则让新节点的next域设为NULL,再让head指向新节点,否则先将新节点指向那个 	节点,		  然后将head指针指向新节点
	4.重复第2步继续建立新节点
插表尾算法:
	1.给头指针head赋值NULL
	2.建立新节点,并把该节点的地址赋给指针变量p,将该节点的next指针设NULL
	3.如果head为NULL,则让head指向新的节点,否则把这个新节点插到最后一个节点的后面
	4.将last指针指向这个新节点
	5.重复执行第2步建立新节点

4.输出链表算法

1.定义指针p,让p指向头节点
2.输出p所指节点的数据
3.使p后移一个节点(即指向next)
4.若p不为空,则循环第2不;p为空,则结束

十四.文件操作

1.文件分类和处理方法

分类: 根据储存介质不同分为- 普通文件(文件名.扩展名) 设备文件(键盘 显示器)
     根据数据组织方式不同- 文本文件 二进制文件
处理方法: 缓冲文件系统 非缓冲文件系统

2.文件类型指针

文件类型结构体FILE
指针定义: FILE *指针名

3.标准输入输出指针

键盘输入 stidin 
显示器输出 stdout
错误输出 stderr

4.文件的打开和关闭

文件的操作信息包括: 文件的名字.路径 操作方式
操作方式常用的有 r只读 w只写 a在文件尾部追加 t打开文本文件 b打开二进制文件 +
*其中+的用法: r+先读后写 w+先写后读 a+先追加后读取*
文件说明符: 路径(正常的\变为\\)+文件的名字.扩展名
1.打开方法(fopen 返回值为对应文件结构体的首地址)
  文件结构体类型指针变量=fopen("文件说明符","操作方式")
2.关闭方法(fclose)
  fclose(文件结构体类型指针) 关闭成功返回值为0 否则为非0的值

5.顺序读写函数(前提是使用对应指针已经打开了一个文件)

1.读字符函数-> 返回一个字符,并将指针后移一个字节

用法: 字符变量=fgetc(文件结构体类型指针)
	读取错误时返回标志EOF(-1)
	可以用feof()避免识别二进制中的-1
 		用法: 作为循环的条件 !feof(文件指针)

2.写字符函数-> 写成功返回字符 否则为EOF

用法: fputc(字符,文件指针)

3.读字符串函数fgets()

读取n-1个字符在字符串末尾加\0,每当读取的\n时结束并将指针后移,同样多个字符单位
用法: fgets(字符数组变量,n,文件指针)

4.写字符串函数-> 写入时不包括\0

用法: fputs(字符串,文件指针)

5.格式化读取函数

用法: fsacnf(文件指针,格式控制,&输入变量表列)
字符串变量前不用加&

6.格式化写入

用法: fprintf(文件指针,格式控制,输出变量表列)

7.数据块读取

用法:fread(新数据地址,n,数据项数目,文件类型指针)
作用:读取文件指针中指定个数数据项,每个数据项读写n个字节数,并将数据存入新地址

8.数据块写入

用法:fwrite(新数据地址,n,数据项数目,文件类型指针)
作用:在文件指针中写入指定个数数据项,每个数据项写入n个字节数,并将数据存入新地址

6.随机(定位)读写函数

1.文件读写位置指针
2.位置指针重返文件头函数
  调用格式: rewind(文件类型指针)
3.改变文件位置指针函数
  调用格式: fseek(文件类型指针,偏移量,起始点)
    起始点参数: SEEK_SET或0 ->开始位置
			  SEEK_CUR或1 ->当前位置
			  SEEK_END或2 ->末尾
4.查询文件位置指针
  调用格式: ftell(文件类型指针)

十五.位运算和可变数组

1.位运算符: 二进制数运算 如 01011101 位运算符 10111101

逻辑位: & 与:两位都为1则该位为1,否则为0
       | 或 :两位同为0则该位为0,否则为1
       ^ 异或:两位不同时为1,相同时为0   
             规定: x^x=0 0^x=x x^y^y=x
       ~ 取反:各位的0变成1，1变成0

2.定义数组结构体

typedef struct{int *array;int size;}Array;

3.对应操作函数

//创建
Array arrayCreat(int array_size){
     Array a;
     a.size=array_size;
     a.array=(int*)malloc(sizeof(int) * a.size);
     return a;

}
//释放
void arrayFree(Array *a){
    free(a->array)
    a->array=NULL;
    a->size=0;
}
//查询
int arraySize(Array *a){
    return a->size;  //返回其数组大小即可
}
//引用
int arrayGet(const Array *a,int index){
    return a->array[index];  //返回其值即可
}
//修改
void arraySet(Array *a,int index,int value){
    a->array[index]=value;
}
//扩大
void arrayExpend(Array *a,int more_size){   //始终实在对原数组进行操作
    int *p=(int*) malloc(sizeof(a->size+more_size));
    for(int i =0;i<a->size;i++){
        p[i]=a->array[i];
    }
    free(a->array);
    a->size +=more_size;
    a->array=p;
}
