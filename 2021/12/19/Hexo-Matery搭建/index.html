一.搭建hexo博客

1.git指令

在指令前加sudo是最高权限
hexo clean清除缓存hexo g重新加载
hexo init初始化
hexo s静态生成本地网页
hexo serve动态生成本地的网页
hexo server -p 4001 (端口号)
hexo new “标题”    生成博客文件
hexo new page "导航菜单名(英文)" 生死导航栏文件 在 Blog\hexoblog\source里
hexo d推送到github    
    如遇到spown failed的错误
    处理办法：将hexoblog中的.deploy_git文件删除再上传
git rm 文件名名称    删除本地文件
git rm -r 文件夹/      删除本地文件夹
npm run 自定义指令字符串   
    自定义指令在/hexoblog/package.json文件的scripts后修改
        格式为 "自定义字符串": “hexo指令(多个指令用&&连接)”

2.利用install.sh搭建hexo script步骤

前提：
    安装好nodejs和git
在Blog文件夹内git
    运行：
        source install.sh hexo_win
    测试：
        source install.sh test_git
        source install.sh test_node
        source install.sh deploy
        (返回版本号即为成功)
然后将up.sh放入hexoblog文件夹
在Blog/hexoblog中git
    下载插件:
        git init
        npm install hexo-deployer-git --save         
        npm install -g hexo         
         npm update hexo -g             #升级hexo
        npm install -S hexo-prism-plugin
        npm install hexo-cli -g
        运行source up.sh

3.git配置SSH key

1.先看本地是否配置好SSH key（命令：cd ~/.ssh）
  ssh配置好之后是保存到 c:/用户/Administrator/.ssh
  若不能找到该文件夹则说明没配置
2.先生成ssh key
  输入生成命令：ssh-keygen -t rsa -C "github账号绑定的邮件地址"
  连续回车四次即可
  进入到ssh文件夹，复制id_rsa.pub文件里全部内容
  接着打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key
  将id_rsa.pub复制的内容粘贴到key中，title随便
  点击 Add SSH key
3.测试是否成功：
    ssh -T git@github.com
    输入测试命令后，接着输入yes
4.配置账号和密码:
  $ git config --global user.name " "     #github用户名
  $ git config --global user.email " "     #填写github注册邮箱

4.新建github仓库

仓库名设置为
用户名.github.io.git
然后更改\hexoblog\_config.yml文件中的设置
    deploy:
          type: git
          repository: https://github.com/用户名/用户名.github.io.git
          branch: main

5.安装图床并更换主题

图床作用：保存博客里的图片到云端，使得写博客时插入的图片链接有效
下载安装picgo并配置图床设置
    1.在github/settings/Devloper settings新建token并勾选repo
    2.写入配置
    3.在picgo里面上传博客需要用到的图片文件即可 然后复制链接在博客文章里使用

6.更换主题

将主题文件夹放入\hexoblog\themes文件夹中，并更改\hexoblog\_config.yml文件中的以下设置
      theme: 主题文件夹名

三.博客配置

1.生成博客

在hexoblog中git输入hexo new “文章标题” 生成博客文件
然后在文件的顶部添加模板，修改模板和简介内容即可

2.博客模板

---
title:                  #标题
date:                     #日期    
author: 乄尘封               #作者
img:                   #预览图的路径
coverImg:               #在首页的预览图
top: false                #是否置顶
cover: false            #在首页是否轮番播放
toc: true                    
mathjax: false            #数学公式模块
password:                #访问密码
summary:                 #描述(用双引号包裹)
keywords:                #关键词(用来搜索)
tags: blogs                    #标签(用[]包裹，多个标签用，号隔开)
categories:             #分类
---
(首尾用---包括,位于博客文件的顶端)

1.主页的遮幕

在matery.css中的rainbow相关的全部注释掉即可关闭

2.修改主页横幅标题的字体：

将字体文件放入到/hexoflag/source/font文件夹中     
(没有font就新建,字体文件可以在方正字库中下载)
然后在主题中的/css/my.css中设置添加一下代码
    @font-face{
        font-family: '字体';
        src: url('../font/字体.TTF');
    }
    .diyFont{
        font-family: '字体', serif;
    }
然后在\hexoblog\themes\matery-master\layout\_partial\bg-cover-content.ejs
中修改class="title center-align diyFont"
     class="description center-align diyFont"即可
主标题文字在全局配置中 subtitle:
副标题文字在主题全局配置中 subtitle:
